# Diagrams

- **Mermaid** diagrams live inline in the architecture `.md` files and render on GitHub
  and in most Markdown viewers.
- **Excalidraw**: `high-level.excalidraw` is an editable scene. Open at
  https://excalidraw.com (File → Open) or via the Excalidraw VS Code extension. It is a
  starting hand-drawn-style overview meant to be extended as the design evolves.

Keep diagrams in sync with the prose; if a diagram and text disagree, the ADRs in
`../../adr/` are the tie-breaker.
